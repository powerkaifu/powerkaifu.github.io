發表於 2020-08-25 更新於 2020-10-01

字型

CSS 常用的字型設定：

單位

  • px：字型大小以像素為絕對單位。
  • em：字型大小參考父元素為倍數相對縮放尺寸，例如：父元素外層是 10px，子元素 2em 則是 20px。
  • rem：相對於根元素 (:root or html) 的文字尺寸，預設是 16px，元素設定 2rem 則是 32px。

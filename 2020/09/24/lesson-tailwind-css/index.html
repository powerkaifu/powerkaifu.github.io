Tailwind CSS

發表於 2020-09-24

介紹篇

使用過 Bootstrap CSS，我們知道它擁有強大的網格系統與 Components UI 提供設計者使用，另外也有貼心的 Utility 工具調整內容、寬度、距離等等，但在操作切版的自由靈活度上稍嫌不足，以至於設計出的版型常常長得很像，但這不能怪 Bootstrap，它之所以成為全球著名的 CSS Framework 之一，設計的核心是希望被普遍大眾接受，容易使用。

Tailwind CSS 屬於通用工具類型（Utility）的 CSS 框架，Bootstrap 則被歸類為（Component UI），與 Bootstrap 最大的不同是，Tailwind CSS 全力支持使用工具類型的 Class 讓設計者完成客製化的版型，甚至可以不用寫 CSS 或鮮少就能完成版型，在元素標籤內靈活操作 Class。由於 Tailwind 靈活度高，使用的都是 Class 組合技，切版速度更快，能針對客製化符合修改需求，但比起 Bootstrap 更考驗了設計者對於 CSS 的了解。
